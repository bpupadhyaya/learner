# Governance

## Project Model
Learning uses a maintainer-led governance model.

## Roles
- Maintainers: review/merge code, set roadmap priorities, enforce standards.
- Contributors: propose, implement, and discuss improvements.

## Decision Making
- Prefer consensus in issues/PRs.
- If consensus is not reached, maintainers decide based on project goals, risk, and maintainability.

## Changes to Governance
Maintainers may update this document through pull requests with public rationale.
