# Support

## Getting Help
- Questions and usage help: open a GitHub Discussion (recommended) or an Issue.
- Bugs: use the Bug Report template.
- Features: use the Feature Request template.
- Security concerns: use GitHub Security Advisories as described in [SECURITY.md](SECURITY.md).

## Before Opening a Ticket
- Search existing issues/discussions.
- Verify behavior on latest `main`.
- Include reproducible steps and environment details.

## Response Expectations
- Best effort community support.
- Maintainers prioritize correctness and security work.
- Urgent security issues follow [SECURITY.md](SECURITY.md).
